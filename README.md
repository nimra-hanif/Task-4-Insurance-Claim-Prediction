Task Objective: The objective of this project is to predict medical insurance claim amounts using personal and demographic information from the Medical Cost Personal Dataset. This is a regression problem where the goal is to estimate insurance charges based on features such as age, BMI, smoking status, number of children, sex, and region.
Approach: The project followed a structured machine learning workflow: the first step was Data Loading and Understanding: loaded and explored the Medical Cost Personal Dataset and reviewed dataset features, structure, and target variable. The next step was Data Cleaning and Preparation: checked for missing values and duplicate records, encoded categorical variables for machine learning and prepared input features and target variable for modeling. After that in  Exploratory Data Analysis (EDA): visualized relationships between age, BMI, smoking status, and insurance charges and identified important factors influencing medical costs. Moreover, in Model Training: the dataset was split into training and testing sets and Linear Regression model was trained to predict insurance charges. Then came Model Evaluation in which model performance was evaluated using Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE). Furthermore, Results and Insights showed that Model Performance: MAE: 4181.19 and RMSE: 5796.28. Key Insights: Smoking status has a significant impact on insurance charges. Insurance costs generally increase with age. Higher BMI tends to be associated with higher medical expenses. The model produced reasonable predictions, though some larger prediction errors indicate room for improvement. In conclusion, this project demonstrates how machine learning can be applied to estimate medical insurance claim amounts and highlights the importance of demographic and lifestyle factors in predicting healthcare costs.
